Política de Privacidade

Última atualização: 16 de junho de 2026

O aplicativo Carros Mik Dundee é uma ferramenta interna de coordenação logística utilizada exclusivamente durante a Gincana Mik Dundee. Esta política descreve quais dados são coletados, como são usados e os direitos dos usuários.

1. Dados coletados

  • Localização GPS — coletada apenas enquanto o aplicativo está em uso (foreground), para que a Mesa Central visualize a posição dos veículos em tempo real durante o evento.
  • Dados de autenticação — e-mail e senha utilizados para login no sistema. Armazenados de forma segura via Supabase Auth.
  • Dados de missões — registros de solicitações atribuídas, status de atendimento e horários, usados para coordenação operacional durante o evento.

2. Uso dos dados

Os dados coletados são usados exclusivamente para:

  • Exibir a posição dos motoristas no mapa da Mesa Central
  • Atribuir e acompanhar missões durante o evento
  • Autenticar usuários no sistema

Nenhum dado é utilizado para fins comerciais, publicidade ou análise de comportamento.

3. Compartilhamento de dados

Os dados não são vendidos, alugados ou compartilhados com terceiros. O acesso é restrito aos participantes autorizados do evento (motoristas, líderes de mesa e equipe da Mesa Central).

4. Retenção dos dados

Os dados operacionais (localização, missões) são mantidos apenas durante o período do evento. Os dados de autenticação podem ser removidos mediante solicitação ao responsável pelo sistema.

5. Segurança

Os dados são armazenados na plataforma Supabase, com Row Level Security (RLS) ativo em todas as tabelas, garantindo que cada usuário acesse apenas os dados pertinentes ao seu papel. As credenciais são armazenadas de forma criptografada no dispositivo.

6. Direitos do usuário

A qualquer momento, o usuário pode:

  • Solicitar a exclusão de sua conta e dados associados
  • Revogar a permissão de localização nas configurações do dispositivo
  • Solicitar uma cópia dos dados armazenados
